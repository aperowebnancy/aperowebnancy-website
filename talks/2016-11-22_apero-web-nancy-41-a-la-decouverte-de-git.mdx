Vous pensez que GIT est un dépôt de fichiers ? Vous ne voyez pas comment il va vous permettre de bosser en équipe ? Vous pensez qu'il apporte plus d'inconvénients que d'avantages ?

Venez découvrir comment se servir de l'outil de versionning le plus utilisé sur le marché.

Vous allez apprendre comment utiliser GIT de manière efficace, comment profiter de sa puissance pour gérer des situations critiques, comment travailler en équipe distante et pluridisciplinaire.

Au programme :

• créer des branches et travailler avec votre équipe - best practice
• basculer, fusionner, exploiter les branches - merge, rebase, cherry-pick
• pull-request, publication et diffusion de version
• situation critique: comment s'en sortir - push -f, gitk, pointeur

Par BigInt (https://bigint.fr/).
