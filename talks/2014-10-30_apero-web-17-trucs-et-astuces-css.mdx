Votre "vertical-align" refuse de centrer ce fichu texte ? "Flex-box" ne se comporte pas comme vous l'attendez ? Vous ne savez pas comment structurer proprement vos fichiers CSS ? Qu'à cela ne tienne, cet apéro est fait pour vous !

Venez échanger vos meilleures astuces CSS lors de cette 17ème édition des Apéros Web Nancy. Que ce soit un plug-in pour vous faciliter le développement, des retours d'expériences sur l'organisation de vos feuilles de style ou des explications sur le fonctionnement d'une fonctionnalité méconnue, n'hésitez pas à y partager vos connaissances !
