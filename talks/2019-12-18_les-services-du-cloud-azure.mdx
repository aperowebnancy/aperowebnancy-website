Pour décembre, on vous propose un Apéro Web sur les services Azure :

Aujourd'hui les services Cloud sont dominants dans le WEB, dans ces services Azure détiens 14% des parts du marché. Il est le service Cloud qui monte le plus vite, il est aujourd'hui utilisé par plus de 80% des 500 plus grandes entreprises. Le but de cette présentation n'est pas de vous convertir à Azure mais de comprendre pourquoi ce service est si populaire aujourd'hui. Ce n'est pas qu'un simple service d'hébergement, voyons ensemble ce qu'il a à présenter.
