Magento est une solution e-commerce open source dominante sur le marché, basée sur le framework PHP Zend, et développée sur patron d'architecture MVC (Modèle Vue Contrôleur).

Jean-Paul Garcia, qui surfe sur la vibe de Magento depuis maintenant 6 ans, va vous en présenter les fonctionnalités principales ainsi que l'architecture technique mise en oeuvre pour faire fonctionner le "bousin" : modèle, layout, bloc, template, contrôleur...
