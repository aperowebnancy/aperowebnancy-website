Exposer une API GraphQL est relativement simple, notamment grâce au nombre de ressources présentes sur le Web.
Mais, bien peu mentionnent la gestion des autorisations, laissant chaque développeur tâtonner de leur côté, sans réel standard de-facto.
Du moins, c'était le cas avant l'émergence de graphql-shield.
Découvrons ensemble ce module incontournable lors d'une session de live-coding.
