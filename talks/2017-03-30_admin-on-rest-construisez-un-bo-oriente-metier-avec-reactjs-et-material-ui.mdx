Les back-offices (aussi appelés admins), tous les clients en veulent, mais sans vouloir payer pour.

Et quand ils se retrouvent avec un phpMyAdmin, ils font la tête.

Avec les outils d'aujourd'hui (React.js, material-ui), on peut développer une admin orientée métier de qualité, mais ça coûte cher... à moins d'utiliser un générateur d'admin.

C'est l'objet d'admin-on-rest, un projet open-source développé par Marmelab, que [François Zaninotto](https://twitter.com/francoisz) vous présentera.
